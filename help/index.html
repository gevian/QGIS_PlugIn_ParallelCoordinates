Parallel Coordinates for Quantum GIS plugin help

This plugin allows to visually analyze tabular data using the parallel coordinates technique.
By default, the plugin will attach a new window to the bottom of the QGIS application but can be moved to other more convenient locations.
Since this plugin is still under development data loss or other problems may occur. Please, keep this in mind.

Current layer

This combo box at the very lower left of the window allows the selection of one vector layer listed in the table of contents.

Select attributes

When clicking on the "Select attributes" button, the user can select the attributes of the chosen vector layer by checking the corresponding box on the very right.

Hide features not in extent

When checked the parallel coordinates plot will only display the lines of the features being currently visible in the map canvas.
When unchecked all features of the current layer are being displayed.

Help

Displays this help page.

Changing the axis order

Changing the position of an axis can be achieved by dragging it to another position.
If this position is another axis, both axes will switch their positions.
If this position lies exactly between two axes, the dragged axis will be moved to the position between these two axes.

Selecting features of interest

Features of interest can be selected by dragging a rectangle around a part of an axis. The selection then is highlighted in red color.
Changing the selection using the map canvas or the attribute table also changes the selection of the parallel coordinates plot.
